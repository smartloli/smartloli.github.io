Configuration

> Installation > Configuration

Overview

Kafka Eagle has a number of configuration options that you can specify in a .properties configuration file or specified using environment variables.

Note:
You must restart Kakfa Eagle for any configuration changes to take effect.

Config file locations

Do not rename system-config.properties! Kafka Eagle defaults are stored in this file.

Base Common

Name Default Describe
kafka.eagle.zk.cluster.alias cluster1,cluster2 Kafka multi cluster alias attribute.
cluster1.zk.list ke01:2181,ke02:2181,ke03:2181/cluster1 Kafka cluster1 zookeeper address.
cluster2.zk.list ke01:2181,ke02:2181,ke03:2181/cluster2 Kafka cluster2 zookeeper address.
kafka.zk.limit.size 25 Kafka Eagle maximum number of connections for the Zookeeper client.
cluster1.kafka.eagle.broker.size 10 Kafka broker size online list.
cluster2.kafka.eagle.broker.size 20 Kafka broker size online list.
kafka.eagle.webui.port 8048 Kafka Eagle webui port.
cluster1.kafka.eagle.offset.storage kafka Kafka offsets stored in kafka.
cluster1.kafka.eagle.offset.storage zk Kafka offsets stored in zookeeper.
kafka.eagle.metrics.charts false Kafka Eagle default disable metrics.
kafka.eagle.metrics.retain 30 Kafka Eagle default retain metrics data.
kafka.eagle.sql.fix.error false Kafka Eagle default disable fixed kafka sql query error.
kafka.eagle.sql.topic.records.max 5000 Kafka Eagle SQL query topic max records.
kafka.eagle.topic.token keadmin Kafka Eagle delete topic token.
