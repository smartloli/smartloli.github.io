Getting Started

> Introduce > Getting Started

Overview

This guide will help you get started and acquainted with EFAK. To learn more about EFAK in general, refer to What is EFAK?

Install EFAK

This step varies according to your computer operating system. Refer to the instructions for your OS in the Installation section for instructions.

Log in for the first time

  1. Open your web browser and go to http://localhost:8048/. 8048 is the default HTTP port that EFAK listens to if you haven’t configured a different port.
  2. On the login page, type admin for the username and password.
  3. Change your password.

Note

We strongly encourage you to follow EFAK best practices and change the default administrator password. Don’t forget to record your credentials!
