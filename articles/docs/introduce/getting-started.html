Getting Started

> Introduce > Getting Started

Overview

This guide will help you get started and acquainted with Kafka Eagle. To learn more about Kafka Eagle in general, refer to What is Kafka Eagle?

Install Kafka Eagle

This step varies according to your computer operating system. Refer to the instructions for your OS in the Installation section for instructions.

Log in for the first time

  1. Open your web browser and go to http://localhost:8048/ke. 8048 is the default HTTP port that Kafka Eagle listens to if you haven’t configured a different port.
  2. On the login page, type admin for the username and password.
  3. Change your password.
Note:
 We strongly encourage you to follow Kafka Eagle best practices and change the default administrator password. Don’t forget to record your credentials!
